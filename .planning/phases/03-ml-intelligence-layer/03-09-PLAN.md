Create `src/discord_bot/cog_ml.py` with an `MLInsightsCog`:

**Slash commands:**

1. `/regime` — show current market regime:
   - Calls regime_manager.get_current_regime()
   - Embed: regime state (color-coded: green=risk-on, yellow=risk-off, red=crisis), probability, expected duration, transition matrix summary
   - Chart: regime state timeline (last 30 days) — use matplotlib bar chart with color-coded states

2. `/forecast` — show volatility forecast:
   - Calls vol_manager.get_current_forecast()
   - Embed: 1-day and 5-day vol forecasts, current IV rank, RV/IV spread (variance risk premium), Hurst exponent
   - Interpretation: "Vol is [overpriced/underpriced/fair]. Entry timing [favorable/unfavorable] for premium selling."

3. `/sentiment` — show market sentiment:
   - Calls sentiment_manager.get_current_sentiment()
   - Embed: sentiment score (-1 to 1), velocity, positive/negative/neutral breakdown
   - Include contrarian signal flag: if |sentiment| > 0.6 AND PCR > 1.2 → "Contrarian signal active"

4. `/anomalies` — show current anomaly scan:
   - Calls anomaly_manager.get_current_anomalies()
   - Embed: overall score (color: green <0.4, yellow 0.4-0.7, red >0.7), top anomaly flags, top strike clusters
   - Include flow summary if available (sweep/block counts, net premium)

5. `/reasoning` [optional ticker] — run Claude reasoning analysis:
   - Assembles full MarketContext
   - Calls reasoning_manager.run_analysis()
   - Embed: summary, regime assessment, vol outlook, signal conflicts, strategy recommendations, risk warnings
   - This is the most expensive command (~$0.01-0.03 per call) — add a 5-minute cooldown

6. `/ml_health` — show model health dashboard:
   - Calls learning_manager.get_model_health()
   - Embed: accuracy by signal type, calibrated confidence, trend indicators
   - Include data source status (which APIs are configured and working)

Add embed builders to `src/discord_bot/embeds.py`:

- `build_sentiment_embed(sentiment: dict) -> discord.Embed`
- `build_anomaly_embed(report: AnomalyReport) -> discord.Embed`
- `build_reasoning_embed(analysis: dict) -> discord.Embed`

   - Update vol forecast
   - Update sentiment
   - Run anomaly scan
   - Log all results to feature store
